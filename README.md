# Titanic Survival Predictor 🚢

## 🌐 Live Demo
👉 [Try the app here!](https://titanic-predictor-cv9dw9ybikrrdpziwvyzzf.streamlit.app/)

A machine learning web app that predicts whether a 
Titanic passenger would have survived based on their 
details.

## 🎯 Problem Statement
Predict survival of Titanic passengers based on 
passenger class, gender, age and fare paid.

## 🛠️ Tech Stack
- Python
- Scikit-learn (Logistic Regression)
- Streamlit
- Pandas, NumPy

## 📈 Model Performance
- Accuracy: 80.45%
- Algorithm: Logistic Regression
